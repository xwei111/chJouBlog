# 前言

最近chatGPT那可真是火的一塌糊涂，你敢问他就敢答，而身为程序员更是对他的编码能力感兴趣，所以赶紧去体验了一波，没想到想要体验一波高科技也不是那么容易的，各种限制，对想学习的人来说是真的不够友好啊

这里有一篇关于关于chatGPT的介绍 (opens new window)可以先看一下

# 准备

  2. 最后，手机（有限制，但也有办法）

# 谷歌插件

大家可以先下载goole插件，前往 (opens new window)，下载chromium.zip，将其放入google的扩展程序（打开开发者模式），然后再google中搜索内容，结果如下

搜索完毕右侧提示需要登陆，点击前往登录，登录需要邮箱，自行注册一个gemail邮箱即可，注册完毕邮箱注册chatGPT账号，注册完毕登录即可，

# 手机验证

初次登录需要手机号验证，但是国内手机号是不行的，接下来介绍一下如何获取验证码

# 获取验证码

sms-activate.org (opens new window)，这个网站可以帮助大家通过手机验证的通过，进入此网站，先注册账号，注册完毕后选择余额 - 充值 - 支付宝，充0.3美元即可（大概2rmb），冲多了也没用，如下

充值完毕后就可以选择对应的国家、服务，购买即可，我第一次买的印度，但是发送验证码时提示是固定电话，无法发送验证码，后改为英格兰后ok

购买完毕后如下

赋值手机号前往验证，发送验证码后在此页面刷新几次看到验证码即可
